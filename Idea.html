PAPER 01
★ High Priority IEEE S&P / CCS Novel Arch
HyperMamba-NIDS: Continuous-Time Heterogeneous Hypergraph Mamba for Multi-Vector Intrusion Detection
Your flagship paper. Fuses Graph-Mamba's linear-time SSM with hyperedge modeling to detect coordinated attacks (botnets, DDoS) that pairwise GNNs structurally cannot capture.
Methods from your surveys
GraMO / SAMBA k-HNN Hypergraph FactorGCL residual CTDG formulation Hyper-GCN
Target Dataset
CICIDS, IoT-23, UNSW-NB15
Core Novelty
First Hyper-Mamba for security
Complexity
O(N) temporal + O(E·k) spatial
Traffic Type
Standard + IoT
Key contributions
  • First architecture to model coordinated multi-host attacks as hyperedges in a continuous-time dynamic graph
  • Bidirectional selective SSM encoder replaces quadratic Temporal Graph Transformer — O(N) scaling
  • Hyperedge-centric random walk tokenizer to serialize hypergraphs into Mamba-compatible sequences
  • Evaluated on standard, enterprise, and IoT benchmarks — broadest coverage in NIDS GNN literature
  • Directly addresses Gap 3 from your Graph ML survey (SSM + Hypergraph unification)

PAPER 03
★ High Priority NeurIPS / ICML Novel Arch
SheafNIDS: Bayesian Sheaf Neural Networks for Heterophilic Network Anomaly Detection with Uncertainty Quantification
Network traffic is deeply heterophilic — benign and malicious flows share the same hosts. BSNN's orthogonal restriction maps and Bayesian UQ are perfectly suited; no prior NIDS paper has used sheaf theory.
Methods from your surveys
Bayesian Sheaf NN (BSNN) Cayley distributions SO(n) Variational inference (ELBO) Sheaf Laplacian
Target Dataset
NSL-KDD, UNSW-NB15, CIC-IDS
Core Novelty
First sheaf-based NIDS model
Key Strength
Uncertainty scores per alert
Traffic Type
Standard + Enterprise
Key contributions
  • Model NIDS as a heterophilic graph classification problem — attack nodes connect to benign neighbors
  • Sheaf Laplacian restriction maps allow per-edge geometric rotation, preserving distinct node identities
  • Bayesian posterior over sheaf structure gives calibrated confidence per detection — critical for SOC analysts
  • Robustness to adversarial evasion (noisy/perturbed features) via Bayesian priors
  • Addresses Gap 1 from Graph ML survey — extend to stochastic base graph for fully uncertain topologies

PAPER 05
Mid Priority NeurIPS / ICLR Novel Arch
MotifIDS: Beyond Message Passing — Attack Signature Detection via Neural Graph Pattern Machines
Attack signatures are topological motifs (port scan patterns, C2 beaconing, lateral movement triads). GPM's motif tokenizer directly encodes these without message passing, eliminating over-smoothing on traffic graphs.
Methods from your surveys
Neural GPM (ICML 2025) Random walk tokenizer Cross-attention motif ID GRU/LSTM pattern encoder
Target Dataset
CICIDS, NSL-KDD
Core Novelty
Attack = motif vocabulary
Key Strength
Interpretable attention weights
Traffic Type
Standard
Key contributions
  • Define a "Network Attack Motif Vocabulary" — encode known attack patterns as canonical subgraph tokens
  • Differentiable adaptive tokenizer (Gap 2 from Graph ML survey) using RL agent for rare-motif mining
  • Cross-attention weights directly interpretable as "which traffic pattern triggered this alert"
  • Strong OOD generalization — novel attack variants share motif substructures with known attacks

TopoNIDS: Topological Deep Learning for Detecting Network Topology Reconfiguration Under Attack
Applies persistent homology and Ordered TDL to track how the "shape" of a network topology changes during an attack — worms and ransomware reorganize network structure before payloads deploy.
Methods from your surveys
Topological Deep Learning Ordered TDL (2025) Persistent Homology Betti numbers β0/β1/β2 TopoBench framework
Target Dataset
CICIDS, IoT-23
Core Novelty
Attack = topological event
Key Strength
Early detection before payload
Traffic Type
Standard + IoT
Key contributions
  • First use of persistent homology filtration on network traffic graphs for pre-attack detection
  • Betti number time series as early-warning signal — topology changes before traffic volume anomalies
  • Ordered TDL maps directional contagion: how attack propagates directionally hop-by-hop
  • Evaluated on worm propagation and ransomware lateral movement scenarios
PAPER 07
Mid Priority TIFS / TDSC
IrregularFlow-NIDS: Hypergraph Neural Networks for Asynchronous IoT Traffic Intrusion Detection
IoT devices produce deeply asynchronous, irregular traffic. HyperIMTS's irregularity-aware message passing eliminates artificial padding, preserving real attack timing signatures in constrained IoT environments.
Methods from your surveys
HyperIMTS (ICML 2025) Irregularity-aware msg passing Temporal + variable hyperedges Hyper-GCN adaptive topology
Target Dataset
IoT-23, N-BaIoT, MQTTset
Core Novelty
No padding — raw IoT timing
Key Strength
Resource-efficient inference
Traffic Type
IoT / Edge
Key contributions
  • Map each IoT packet event to a node — temporal hyperedges encode device timing, variable hyperedges encode device clusters
  • Irregularity-aware message passing natively handles Mirai-style bursty botnet coordination
  • No padding means no timing signal destruction — attack timing patterns preserved
  • Lightweight enough for edge deployment on resource-constrained IoT gateways

Mid Priority TNNLS / TDSC
ImbalanceNIDS: Contrastive Pre-training on Dynamic Traffic Graphs for Extreme Class Imbalance
NIDS datasets suffer from extreme imbalance (attack traffic <0.1% in real networks). Adapts CRP-AML's contextual risk profiling and GCPAL's contrastive pre-training to maintain F1>80% on minority attack classes.
Methods from your surveys
CRP-AML framework GCPAL contrastive pretraining CLDG temporal invariance FC2L ACOP mechanism
Target Dataset
CICIDS, real enterprise logs
Core Novelty
Security-context risk profiling
Key Strength
Robust at 0.001% attack ratio
Traffic Type
Standard + Enterprise
Key contributions
  • Embed NIDS "red flags" (MITRE techniques, CVE patterns) directly into graph learning pipeline as contextual priors
  • Temporal contrastive objective: same host's normal behavior should be invariant across time windows
  • Deviations from baseline profile weighted as contextual risk, not just statistical anomaly
  • Demonstrate F1 maintenance at 0.0005 attack ratio — matching CRP-AML's AML results

Extended Work NeurIPS / ICLR Survey Gap
DeltaFlow: Gated DeltaNet Linear Attention for Streaming Packet-Level Intrusion Detection
Replaces softmax attention in Temporal Graph Transformers with Gated DeltaNet's targeted memory update — eliminating KV cache explosion for real-time high-speed packet stream analysis.
Methods from your surveys
Gated DeltaNet (ICLR 2025/26) Higher-Order Linear Attention Delta update rule GraMO spatiotemporal
Target Dataset
CAIDA, PCAP-level datasets
Core Novelty
Streaming packet analysis O(1) memory
Key Strength
100Gbps line-rate capable
Traffic Type
All (backbone/ISP level)
Key contributions
  • Gated DeltaNet's targeted write replaces full KV cache — O(1) memory regardless of sequence length
  • Selective forgetting of benign traffic, selective retention of anomalous flow sequences
  • Real-time streaming inference — no window buffering, processes packets as they arrive
  • Designed for high-speed backbone traffic (100Gbps+) where no existing GNN operates
